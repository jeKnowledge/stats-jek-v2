# gitDashboard
A Git web dashboard applied to our git project repository at [DEI Gitlab](https://git.dei.uc.pt)

## Run the project with the following commands:
-  ```meteor run --settings settings.json```

## Requirements
- **nodeJs**
- **Meteor**

## Additional Meteor Modules
- http
